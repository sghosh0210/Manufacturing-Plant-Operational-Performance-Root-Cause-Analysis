# Manufacturing Plant Data Analysis

## Project Overview
This project analyzes operational data from a manufacturing plant to identify the major factors affecting production efficiency, machine performance, quality, downtime, and maintenance. The analysis focuses on finding business insights and providing data-driven recommendations for improving plant operations.

---

## Objective
The main objectives of this project are to:

- Analyze machine performance and downtime.
- Identify high-risk machines.
- Evaluate maintenance effectiveness.
- Analyze product quality and defect rates.
- Compare shift performance.
- Study the impact of environmental conditions.
- Provide business recommendations based on the findings.

---

## Dataset
The dataset contains **3,000 production records** collected from **January 2018 to March 2026**.

It includes information such as:

- Machine ID
- Production Hours
- Downtime Hours
- Maintenance Hours
- Units Produced
- Defects
- Scrap Rate
- Product Type
- Shift
- Operator Count
- Temperature
- Humidity
- Energy Consumption
- Labor Cost
- Rework Hours

## Key Insights

- A few machines consistently experienced higher downtime than others.
- Increased maintenance hours did not significantly reduce downtime.
- Defect rates were similar across all product types.
- Shift performance showed only minor differences.
- Temperature and humidity had little impact on production quality.
- Increasing operator count did not noticeably improve production output.

---

## Business Recommendations

- Prioritize inspection of high-risk machines.
- Move from fixed maintenance schedules to condition-based maintenance.
- Optimize workforce allocation based on production requirements.
- Reduce rework through process improvement.
- Continuously monitor machine performance using KPIs.
